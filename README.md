# Bugtracker Server

REST API for the [bugtracker app](https://www.bugtracker.tk/).\
This app was built using .NET Core 5, and MongoDB. Hosted on Heroku.

## Features

- JWT Authentication
- Role Based Authorization.
- Encrypted user details.
- Users can form groups.
